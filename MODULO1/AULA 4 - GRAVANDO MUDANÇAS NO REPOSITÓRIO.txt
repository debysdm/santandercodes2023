ESTADOS DO GIT
Os possiveis estados de classificação do git
	Untracked - 
	Unmodified - Arquivo mapeado, salvo no estado que está.
	Modified - Arquivo modificado
	Staged - Arquivo será preparado para o "commit"

COMANDOS
git status
	//mostra o estado dos arquivos
git add . ou git add (tab)
	//Irá adicionar o arquivo para ser comitado
